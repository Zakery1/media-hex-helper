# What is this?

Get the colors of popular media platforms

# Installation 

`npm i media-hex-helper --save`

Then...

```
import { mediaHexHelper } from 'media-hex-helper';

mediaHexHelper({
    media_type: 'apple'
})
```

## Options

Media Friend supports 1 option:

* *media_type* - _apple | none_ - (Defaults to none)
